# sTripKaka

A cinematic travel journal with immersive storytelling, dynamic galleries, chatbot guidance, and admin-managed trip content.

## Project Overview

sTripKaka is a featured live project that demonstrates advanced frontend engineering and storytelling-focused UX design. The application provides an immersive travel journal experience with dynamic content management.

## Key Features

- Interactive travel stories with cinematic presentation
- Dynamic gallery system for photo collections
- Chatbot guidance for user navigation
- Admin-managed trip content
- Responsive design across all devices

## Technologies Used

React, TypeScript, Vite, Framer Motion
